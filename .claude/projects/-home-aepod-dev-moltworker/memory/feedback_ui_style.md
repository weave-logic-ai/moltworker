The UI should follow the Track changelog template aesthetic, NOT a dashboard/bento grid look.

**Why:** Current mockups are too boxy with visible rectangular spaces everywhere. On mobile this looks cluttered. Track template is clean, vertical, with generous whitespace.

**How to apply:**
- Start from Track changelog aesthetic: vertical flow, timeline-centric, generous whitespace
- LESS obvious rectangular cards/boxes on mobile
- More flowing, content-first layout
- Sections separated by whitespace and subtle dividers, not heavy card borders
- Timeline is the core interaction pattern for workflows
- Cards should feel light — borderless or very subtle borders
- Content breathes — not packed into grids
- Reference: shadcnstudio.com/templates/track-changelog-template-free
- Flow SaaS template for dashboard elements when needed
- Homepage grid is fine but cards should be lighter/less boxy
